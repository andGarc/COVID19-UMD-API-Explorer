# COVID-19 World Survey Data API Explorer

Explorer app to visualize the indicator results from the [COVID-19 World Symtoms Survey](https://covidmap.umd.edu/)

## Usage
1. Pick a Start Date
2. Pick and End Date
3. Select and Indicator
4. Select countries
